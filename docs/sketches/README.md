# Sketches

This directory holds live exploratory notes that are still informing active
planning or follow-on design decisions.

## Use this folder for

- early design work that is not yet stable enough to become an active plan;
- follow-on design notes that active plans still reference directly; and
- research inputs that remain unresolved.

## Move a sketch out of the live folder when

- the work is implemented and the durable record already lives in
  `docs/plans/completed/`;
- a later active plan supersedes the sketch as the execution surface; or
- the sketch is still worth preserving, but only as historical context.

## Locations

- Keep live sketches here.
- Move superseded or implemented sketches to `archive/`.
- Move completed reviews and remediation source records to
  `docs/plans/completed/` instead of leaving them here.
